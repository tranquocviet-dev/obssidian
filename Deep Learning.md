A sub section of [[Machine Learning]], using multi-layered neural network to do tasks such as classification, regression, representation. Basically just Machine Learning with more layers.

Deep learning has 3 sections:
- Input Layer receives the raw data
- Hidden Layer performs calculations and transformations on the data
- Output Layer produces the final prediction or output of the model
