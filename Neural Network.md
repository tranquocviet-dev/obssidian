Neural Network in [[Machine Learning]] contain different sections:
- Neurons are the basic computation unit
- Weight represent the strength of connection between neurons
- Activation Function aims to make the machine learn more complex patterns
- Bias value is a value to control the output
- Loss function shows the difference between predicted output and reality, a.k.a a performance metric
- Output is the value we want to find
